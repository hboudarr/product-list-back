## Dependencies

- express
- express formidable
- cors
- mongoose


## Setup instructions

Clone this repository 

```
git clone https://github.com/hboudarr/product-list-back.git 
```

Install dependencies

```
npm install
```

Run it

```
npx nodemon index.js
```
